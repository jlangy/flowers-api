About

This is a machine learning algorithm built to identify the species of uploaded flower images. To use, upload an image of a flower from your computer using the browse button, and click predict to see the algorithm's predictions. You will be able to see the top predictions along with the confidence of each prediction as a percentage.

This algorithm was trained on a dataset containing the species listed below, and will not recognize other species. It was built using pytorch, and the training implementation can be seen here.
